# 单点登录 (SSO)

单点登录功能允许企业员工通过企业自有的身份认证系统直接登录腾讯电子签控制台。

## 1. 核心概念
- **IDP (Identity Provider)**：您的企业身份认证系统。
- **SP (Service Provider)**：腾讯电子签。

## 2. 核心接口
- **CreateSingleSignOnEmployees**：创建支持单点登录的员工。
- **DescribeSingleSignOnEmployees**：查询单点登录员工信息。
- **DeleteSingleSignOnEmployees**：删除单点登录权限。

## 3. 配置流程
1. **元数据交换**：在电子签控制台上传您的 SAML 或 OIDC 元数据。
2. **员工同步**：将企业员工的标识（如 Email 或 OpenId）同步至电子签系统。
3. **跳转访问**：员工在企业门户点击电子签图标，通过 SSO 协议自动完成登录。

## 4. 优势
- **账号统一管理**：员工离职后，企业系统禁用账号即可自动阻断电子签访问。
- **免密登录**：提升员工使用体验。
- **安全性**：基于标准的安全协议（SAML 2.0 / OIDC）。
