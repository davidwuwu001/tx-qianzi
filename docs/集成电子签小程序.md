# 集成电子签小程序

腾讯电子签小程序提供便捷的移动端签署体验，支持多种跳转和集成方式。

## 1. 跳转方式 (EndPoint)
- **HTTP**：通过生成的 `http_url` 直接在浏览器或微信外打开。
- **APP**：通过小程序 `path` 在微信内部或其他小程序中跳转。

## 2. 核心接口
- **CreateSchemeUrl**：获取跳转至电子签小程序的签署链接。
- **CreateMiniAppPrepareFlow**：创建小程序发起合同的预处理链接。

## 3. 业务场景
### 场景一：主动通知签署
通过短信或微信发送链接，用户点击后直接进入小程序对应合同详情页。
- **关键参数**：`PathType=1`, `FlowId`。

### 场景二：小程序间跳转
在您的小程序中，通过 `wx.navigateToMiniProgram` 跳转到电子签小程序。
- **AppId**: `wxd2ad33560764654b` (正式环境)
- **Path**: 获取到的接口返回路径。

## 4. 优势
- **无需开发**：直接使用电子签成熟的签署 UI。
- **安全合规**：内置腾讯慧眼人脸核身，确保身份真实。
- **多端同步**：签署结果实时同步至 PC 控制台。
