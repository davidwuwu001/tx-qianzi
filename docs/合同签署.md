# 合同签署

腾讯电子签支持多种签署方式，包括小程序签署和 H5 页面签署，并提供撤销流程的能力。

## 1. 获取小程序签署链接
用于主动通知签署方，提供对应签署链接，在电子签小程序签署。支持跳转到小程序主页、合同详情页或合同列表页。

详细参数请参考：[获取小程序签署链接](file:///d:/Project/tx-qianzi/docs/%E8%8E%B7%E5%8F%96%E5%B0%8F%E7%A8%8B%E5%BA%8F%E7%BD%B2%E5%90%8D%E9%93%BE%E6%8E%A5.md)

## 2. 获取 H5 页面签署链接
腾讯电子签支持可嵌入的 H5 页面，帮助开发者解决实际业务中的移动端集成的场景。H5 页面可以灵活的嵌入到 H5、App 中，在您的应用中完成签署功能闭环。

### 操作步骤
1. **根据合同 ID 获取合同签署链接**：开发者需要将个人签署方的信息作为接口参数传递。
2. **访问签署链接：预览合同**：将返回的页面链接，放到页面的 iframe 中。
3. **签字并进行人脸识别完成签署**。

详细文档：[获取H5签署链接](file:///d:/Project/tx-qianzi/docs/%E8%8E%B7%E5%8F%96H5%E7%BD%B2%E5%90%8D%E9%93%BE%E6%8E%A5.md)

## 3. 撤销签署流程
如果某个合同流程当前至少还有一方没有签署，则可通过该接口取消该合同流程。常用于合同发错、内容填错，需要及时撤销的场景。

详细文档：[撤销签署流程](file:///d:/Project/tx-qianzi/docs/%E6%92%A4%E9%94%80%E7%BD%B2%E5%90%8D%E6%B5%81%E7%A8%8B.md)
