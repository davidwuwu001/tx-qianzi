# 线上环境配置

当您的应用在联调环境测试通过后，需要切换到线上正式环境。

## 1. 切换步骤
1. **更换密钥**：将 `SecretId` 和 `SecretKey` 替换为线上环境的密钥。
2. **更新域名**：
    - 通用接口：`ess.tencentcloudapi.com` 或 `essbasic.tencentcloudapi.com`
    - 文件服务：`file.ess.tencent.cn`
3. **更换 AppId**：确保使用的是线上正式创建的应用 ID。

## 2. 检查项
- **企业实名**：确保线上企业已完成实名认证。
- **额度检查**：确保线上账户有足够的合同扣费额度。
- **回调配置**：在线上环境控制台配置正式的回调接收地址。

## 3. 域名参考表
| 环境 | 通用接口 EndPoint | 文件服务 EndPoint |
| :--- | :--- | :--- |
| 联调环境 | `essbasic.test.ess.tencent.cn` | `file.test.ess.tencent.cn` |
| 线上环境 | `ess.tencentcloudapi.com` | `file.ess.tencent.cn` |
