# 联调环境配置

在应用的开发阶段，推荐使用测试环境的企业进行开发和联调。

## 1. 创建自建应用（联调环境）

1. 打开测试环境控制台：https://beta.qian.tencent.cn
2. 点击"更多" → "自建应用集成"
3. 在左侧选择"应用集成"，可以看到"企业版应用"
4. 点击"测试联调"按钮

## 2. 应用参数配置

在弹出的配置界面中：
- **回调地址**：配置可公网访问的 HTTP POST 接口地址（可以先跳过）
- **回调解密 Key/签名 Token**：点击"系统生成"
- **配置生效**：确认配置后即刻生效

## 3. 获取密钥

- **路径**：【应用集成】 > 【自建应用】 > 【测试联调】 > 【获取密钥】
- **参数**：获取 `SecretId`（AK开头）和 `SecretKey`（SK开头）

⚠️ **重要**：联调环境的密钥与正式环境不同，不是腾讯云 AKID 开头的密钥！

## 4. 联调环境请求地址（企业版）

| 环境地址说明 | EndPoint |
| :--- | :--- |
| 通用接口请求 | `ess.test.ess.tencent.cn` |
| 文件服务接口 | `file.test.ess.tencent.cn` |
