# 错误码

如果返回结果中存在 `Error` 字段，则表示调用 API 接口失败。

## 1. 公共错误码
| 错误码 | 说明 |
| :--- | :--- |
| ActionOffline | 接口已下线 |
| AuthFailure.SignatureFailure | 签名错误，请检查 SecretId/SecretKey 及签名计算过程 |
| AuthFailure.SecretIdNotFound | 密钥不存在 |
| AuthFailure.SignatureExpire | 签名过期（本地时间与服务器时间相差超过5分钟） |
| InvalidParameter | 参数格式或类型错误 |
| MissingParameter | 缺少必选参数 |
| RequestLimitExceeded | 请求频率超过限制 |
| InternalError | 内部错误 |

## 2. 业务错误码
| 错误码 | 说明 |
| :--- | :--- |
| FailedOperation.AgeNotAchieveNormalLegal | 签署人未达到法定签署年龄 |
| FailedOperation.FlowNumExceed | 合同数目超出配额 |
| FailedOperation.NoRoleAuth | 无角色授权 |
| FailedOperation.LicenseNoQuota | 自动签许可额度不足 |
| InvalidParameter.ApproverType | 不合法的签署人类型 |
| InvalidParameter.BizApproverAlreadyExists | 重复添加签署人 |
| ResourceNotFound.Application | 应用号不存在 |
| InternalError.Pdf | PDF 合成错误 |

## 3. 排查建议
- **签名错误**：检查密钥是否正确，注意前后空格，参考官方签名 SDK。
- **参数错误**：对比 API 文档，检查必传字段及数据格式（如 Base64 是否包含头信息）。
- **频率限制**：优化调用频率，或联系技术支持提升配额。
