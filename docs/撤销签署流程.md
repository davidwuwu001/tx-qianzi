# 撤销签署流程

如果某个合同流程当前至少还有一方没有签署，则可通过该接口取消该合同流程。

## 1. 使用场景
常用于合同发错、内容填错，需要及时撤销的场景。
> 注意：如果合同流程中的参与方均已签署完毕，则无法通过该接口撤销合同。

## 2. 撤销签署流程接口
### 接口描述
撤销签署流程

### 使用方法
| 参数名称 | 必选 | 参数类型 | 参数描述 |
| :--- | :--- | :--- | :--- |
| FlowId | 是 | String | 签署流程 ID |
| CancelMessage | 是 | String | 撤销原因，最长 200 个字符 |

### 执行步骤
1. 指定需要撤销的签署流程 ID。
2. 指定撤销合同的原因。
