# 核心页面嵌入

腾讯电子签支持将发起、签署等页面以 H5 的形式嵌入到您的应用或网站中，实现业务闭环。

## 1. 适用场景
- 在您的业务系统中直接打开签署页面。
- 在您的应用内直接发起合同流程。

## 2. 嵌入方式
主要通过获取跳转链接（Url），然后将其放置在 `<iframe>` 或 WebView 中。

## 3. 关键接口
- **CreateFlowSignUrl**：获取 H5 签署链接。
- **CreatePrepareFlow**：创建发起流程的 Web 页面。
- **CreateEmbedWebUrl**：获取其他可嵌入的 Web 页面（如印章管理、合同列表等）。

## 4. 主题配置
您可以通过 `CreateWebThemeConfig` 接口设置嵌入页面的主题色、Logo 等，以匹配您的应用风格。

## 5. 注意事项
- 嵌入页面通常需要处理跨域或安全策略。
- 在移动端 APP 嵌入时，需注意人脸识别的兼容性适配。
