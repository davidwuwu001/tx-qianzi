# 合同预览使用说明

在正式发起合同之前，您可以先生成预览链接，让用户查看合同最终合成后的样子。

## 1. 预览模式开启
在调用 `CreateFlowsByTemplates` 或相关发起接口时，设置参数 `NeedPreview` 为 `true`。

## 2. 预览链接类型 (PreviewType)
- `0`：(默认) 文件流，点开后直接下载预览的合同 PDF 文件。
- `1`：H5 链接，点开后在浏览器中在线展示合同。

## 3. 预览特性
- **不产生合同 ID**：预览模式不会真正创建合同流程，也不会扣减合同额度。
- **有效期**：预览链接通常有效期为 300 秒。
- **不可签署**：预览页面仅供查看，无法进行实名或签署操作。

## 4. 使用场景
- 发起人在提交前最后核对内容。
- 业务系统内部审批流程中查看合同草案。
