# 集成电子签 H5

H5 集成方案允许您将签署功能无缝嵌入到移动端网页、APP WebView 或微信 H5 中。

## 1. 核心流程
1. **获取链接**：调用 `CreateFlowSignUrl` 接口。
2. **嵌入展示**：将返回的 `Url` 放置在 `<iframe>` 或 APP 的 WebView 中。
3. **完成签署**：用户在 H5 页面完成身份验证和签字。

## 2. 关键参数
- **JumpUrl**：签署完成后的重定向地址。
- **Operator**：执行操作的经办人信息。
- **ApproverSignTypes**：指定签署时的认证方式（如仅人脸识别）。

## 3. APP 嵌入适配
在 APP WebView 中嵌入 H5 时，需特别注意：
- **权限申请**：WebView 需要开启摄像头权限。
- **人脸识别**：需按照官方“慧眼人脸核身兼容性文档”进行内核适配，确保人脸流程正常唤起。

## 4. 适用范围
- 仅支持签名控件。
- 仅支持中国大陆身份证和港澳台居民居住证。
- 链接有效期通常为 **30 分钟**。
