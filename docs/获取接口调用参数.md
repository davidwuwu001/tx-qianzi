# 获取接口调用参数

在调用腾讯电子签 API 之前，您需要获取以下核心参数。

## 1. 密钥参数 (SecretId & SecretKey)
- **用途**：用于接口鉴权和识别企业身份。
- **获取路径**：
    - 线上环境：[腾讯云访问管理控制台](https://console.cloud.tencent.com/cam/capi)
    - 联调环境：【企业应用管理】 > 【应用集成】 > 【第三方应用】 > 【测试联调】 > 【获取密钥】

## 2. 应用标识 (AppId)
- **用途**：标识当前调用的应用。
- **获取路径**：在电子签控制台的应用管理界面查看。

## 3. 企业/员工标识 (OpenId)
- **ProxyOrganizationOpenId**：第三方平台子客企业标识。
- **ProxyOperator.OpenId**：第三方平台子客企业中的员工标识。

## 4. 域名参数 (EndPoint)
- **线上环境**：`essbasic.tencentcloudapi.com`
- **联调环境**：`essbasic.test.ess.tencent.cn`
- **文件服务**：`file.test.ess.tencent.cn` (UploadFiles 接口专用)
