# 自动签署使用说明

自动签署（也称静默签）允许企业在无需人工操作的情况下，由系统自动完成合同盖章。

## 1. 适用场景
- 平台方自动盖章。
- 企业内部标准化合同自动签署。

## 2. 配置步骤
1. **创建模板**：在控制台创建好合同模板。
2. **开启服务**：进入【企业设置】->【扩展服务】->【企业自动签署】。
3. **添加模板**：在配置页面添加允许自动签署的模板。
4. **指定印章**：选择该模板自动签署时使用的企业印章。

## 3. 接口调用
在通过 API 发起合同时：
- 指定签署方类型为 `企业静默签`。
- 系统会自动匹配配置好的印章完成签署。

## 4. 注意事项
- **白名单限制**：部分接口的自动签署能力可能需要联系客户经理开通白名单。
- **不支持填写**：自动签署方通常不支持在合同上进行内容填写。
- **PDF 合成**：建议在收到 PDF 合成完成的回调后再进行后续操作。
