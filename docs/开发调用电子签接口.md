# 开发调用电子签接口

腾讯电子签支持多种编程语言的 SDK，推荐使用官方 SDK 进行开发。

## 1. SDK 安装 (以 Node.js 为例)
```bash
npm install tencentcloud-sdk-node-ess
```

## 2. 基础调用流程
1. **实例化认证对象**：传入 `SecretId` 和 `SecretKey`。
2. **配置 HTTP 选项**：设置 `EndPoint`（联调或线上）。
3. **实例化 Client 对象**。
4. **构造请求对象**：填入业务参数。
5. **发起请求并处理响应**。

## 3. 示例代码 (Node.js)
```javascript
const tencentcloud = require("tencentcloud-sdk-node-ess");
const EssClient = tencentcloud.ess.v20201111.Client;

const clientConfig = {
    credential: {
        secretId: "YourSecretId",
        secretKey: "YourSecretKey",
    },
    region: "",
    profile: {
        httpProfile: {
            endpoint: "ess.tencentcloudapi.com",
        },
    },
};

const client = new EssClient(clientConfig);
const params = {
    // 接口参数
};
client.DescribeTemplates(params).then(
    (data) => { console.log(data); },
    (err) => { console.error(err); }
);
```

## 4. 调试工具
- **API Explorer**：支持在线调试、自动生成代码。
- **Postman**：官方提供测试集合，可快速发起调用。
