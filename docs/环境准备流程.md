# 环境准备流程

在正式开始集成腾讯电子签之前，您需要完成以下准备工作。

## 1. 账号准备
- **腾讯云账号**：建议先注册并实名认证一个腾讯云账号。
- **电子签账号**：完成腾讯云账号认证后，需注册腾讯电子签账号。

## 2. 企业认证
- 准备好企业营业执照、法人身份证信息等。
- 在电子签控制台完成企业实名认证。

## 3. 应用创建
- 在电子签控制台创建“自建应用集成”或“第三方应用集成”。
- 获取对应的 `AppId`。

## 4. 权限配置
- 确保操作账号拥有足够的 CAM 权限，建议赋予 `QcloudESSFullAccess` 权限。
- 如果需要使用 API，需要生成 `SecretId` 和 `SecretKey`。
